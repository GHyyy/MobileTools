# MobileTools

> MobileTools 适用平台 Android，是一款针对Android平台开发的测试辅助工具


#### 主要功能：统一设备详情查看入口、网络监测、ping功能、app详情信息查看、测试数据自动导入等

> 初衷：
#### 设备详情：
在移动端测试过程中，经常会遇到查看设备信息：sdk版本、设备IMEI、分辨率、MAC地址、机型、RAM大小等，为便捷查询常见信息，故统一设备详情查看入口
